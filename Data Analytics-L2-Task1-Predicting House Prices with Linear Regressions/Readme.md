# House Price Prediction using Linear Regression

## Overview
This project predicts house prices using a Linear Regression model based on the USA Housing dataset. It demonstrates the complete machine learning workflow, including data preprocessing, exploratory data analysis (EDA), feature selection, model training, prediction, and evaluation.

## Objectives
- Analyze the housing dataset using EDA.
- Identify relationships between features and house prices.
- Build a Linear Regression model.
- Evaluate model performance using MSE, RMSE, and R² Score.
- Visualize predictions and residuals.

## Dataset
- Dataset: USA Housing Dataset
- Target Variable: Price

## Features
- Avg. Area Income
- Avg. Area House Age
- Avg. Area Number of Rooms
- Avg. Area Number of Bedrooms
- Area Population

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Project Workflow
1. Import libraries
2. Load dataset
3. Perform Exploratory Data Analysis (EDA)
4. Handle duplicates and missing values
5. Select relevant features
6. Create correlation heatmap
7. Split data into training and testing sets
8. Train the Linear Regression model
9. Predict house prices
10. Evaluate the model using MSE, RMSE, and R² Score
11. Visualize Actual vs Predicted Prices
12. Perform Residual Analysis
13. Analyze feature coefficients

## Results
The Linear Regression model successfully predicts house prices based on housing-related features. Evaluation metrics and visualization techniques indicate that the model performs effectively and provides meaningful insights into the factors influencing house prices.

## Conclusion
This project demonstrates how Linear Regression can be applied to real-world housing data for price prediction. It provides hands-on experience in data preprocessing, visualization, model building, evaluation, and interpretation.
